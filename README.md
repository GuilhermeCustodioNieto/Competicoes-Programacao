# Programação Competitiva

<img src="logo_obi.png" alt="Logo da OBI" width="200">

Este repositório reúne materiais, exercícios e soluções de problemas utilizados durante meus estudos em **programação competitiva**.

O objetivo principal é desenvolver habilidades de **lógica, algoritmos e estruturas de dados**, visando melhorar o desempenho em competições como a **Olimpíada Brasileira de Informática (OBI)** e a **Maratona de Programação do CPS**.

A programação competitiva é uma área focada na resolução de problemas algorítmicos dentro de limites de tempo e memória, estimulando o raciocínio lógico, a eficiência de algoritmos e o domínio de estruturas de dados. Competições como a OBI incentivam estudantes a desenvolver interesse em ciência da computação por meio de desafios práticos. ([sbc.org.br][1])

---

# Objetivo do Repositório

Este repositório foi criado para:

* Praticar resolução de problemas algorítmicos
* Estudar estruturas de dados e algoritmos

* Preparar para competições de programação
* Organizar materiais de estudo

Os exercícios aqui presentes vêm de diferentes plataformas e simulados utilizados durante a preparação para competições.

# Como Este Repositório Funciona

Este repositório está organizado de forma hierárquica para facilitar a navegação e o estudo progressivo. Ele é dividido em pastas principais que representam fontes de exercícios, anos de estudo e tipos de competições. Cada pasta contém códigos em linguagens como C++, Python e Java, com soluções para problemas específicos.

## Linguagens Utilizadas
- **C++**: Principalmente para problemas que exigem eficiência e manipulação de estruturas de dados complexas.
- **Python**: Para prototipagem rápida e problemas que envolvem manipulação de strings ou cálculos simples.
- **Java**: Usado em algumas competições específicas, como a Maratona CPS.

## Compilação e Execução
Para códigos em C++, utilize a tarefa de build configurada no VS Code (C/C++: g++.exe arquivo de build ativo), que compila o arquivo ativo com flags de debug. Para Python e Java, execute diretamente no terminal ou use interpretadores/compiladores apropriados.

## Estrutura Geral
A organização das pastas segue uma lógica cronológica e temática:

```
Competicoes-programacao/
│
├── arquivos cpp/                 # Arquivos auxiliares em C++

│       ├── Estudo/              # Aprendizado básico

```

# Descrição Detalhada das Pastas

## `arquivos cpp`
Contém arquivos auxiliares em **C++** utilizados para resolver exercícios, incluindo:
- Templates de código para entrada/saída rápida.
- Funções utilitárias para manipulação de vetores, strings e algoritmos comuns.
- Exemplos de estruturas de dados implementadas do zero.

Esses arquivos servem como apoio para a resolução rápida de problemas durante treinos, evitando reescrever código boilerplate.

## `Estudo-2025`
Diretório dedicado aos estudos realizados durante o ano de **2025**. Organizado em subpastas:
- **cpp/2024/**: Revisões e soluções de problemas de 2024, incluindo códigos em C++, Python e Java.
- **cpp/Estudo/**: Aprendizado básico de conceitos como arrays, structs e algoritmos fundamentais.
- **cpp/exercicios-diversos/**: Coleção de exercícios variados, incluindo:
  - Algoritmos de busca e ordenação.
  - Uso de containers como vectors, maps e sets.
  - Problemas de plataformas como Codeforces e OBI.

Essa pasta é usada para prática diária e aprimoramento de habilidades.

## `Kaique-Palestra`
Materiais relacionados a uma palestra ou treinamento sobre programação competitiva ministrada por Kaique. Contém:
- Exemplos de problemas resolvidos.
- Códigos demonstrativos em Python.
- Materiais de apoio para entender conceitos avançados.

Ideal para revisões rápidas de tópicos específicos.

## `Maratona-CPS`
Problemas e exercícios voltados para a **Maratona de Programação do CPS**. Inclui:
- Soluções em Java para problemas como Alfabeto, Concatena, Cubo, etc.
- Foco em algoritmos de manipulação de strings e lógica combinatorial.

Essa pasta prepara para competições em equipe, enfatizando eficiência e colaboração.

## `NeepsAcademy`
Contém exercícios resolvidos da plataforma **Neeps Academy**, uma ferramenta de treinamento para programação competitiva. Os problemas abrangem:
- Cálculos matemáticos e lógicos.
- Manipulação de entrada/saída.
- Algoritmos básicos como soma, média e verificação de condições.

Todos os códigos estão em Python, facilitando a prototipagem.

## `OBI-oficial`
Problemas oficiais da **Olimpíada Brasileira de Informática (OBI)**, organizados por ano e fase:
- **2024/Fase 1/**: Problemas iniciais como concurso, jogo, ogro e relógio.
- **2024/Fase 2/**: Desafios mais avançados como alfabeto, concatena e cubo.
- **2025/fase-2/**: Problemas recentes como distinto, feira, mania e placar.
- **2025/fase1/**: Exercícios básicos como barras, cafeteria, dieta e festa.

Soluções em múltiplas linguagens, com foco em algoritmos eficientes.

## `Simulados-OBI`
Simulados da OBI de anos anteriores, organizados por fase e ano:
- **fase1-2020 a fase1-2023**: Problemas de nível iniciante, como acelerador, camisetas, idade, etc.
- **fase2-2020 a fase2-2024**: Desafios intermediários, incluindo cálculo, câmeras e viagem.
- **fase3-2021**: Problemas avançados como casamento e cubo.

Usado para simular condições reais de competição.

# Como Contribuir ou Usar Este Repositório

- **Para Estudar**: Navegue pelas pastas por tema ou dificuldade. Leia os códigos e tente resolver os problemas antes de ver as soluções.
- **Para Praticar**: Escolha um problema, implemente sua solução e compare com a existente.
- **Para Competir**: Use os simulados para treinar sob tempo limitado.
- **Configurações**: As pastas `.vscode` contêm tarefas e configurações para build em C++.

Se você encontrar erros ou quiser adicionar soluções, sinta-se à vontade para abrir issues ou pull requests.
